# Active-Directory-Homelab

# Purpose

This project stimulates a real-world domain environment using virtual machines(VM):
  Skills Demonstrated
  - User/Group Management
  - Organizational Units(Ou)
  - User Account Unlock
  - User Password Reset

- Windows Server 2022
- Active Directory Domain Services
- VMware
